เปลี่ยนรหัสผ่านผู้ดูแล

      

        
รหัสผ่านเก็บเป็น SHA-256 ในไฟล์ admin.html พิมพ์รหัสใหม่เพื่อสร้าง hash แล้วนำไปวางแทนค่า PW_HASH จากนั้น deploy ใหม่

        

        
        

        
      

    


    

      
🛡️
สรุปสถานะความปลอดภัย (อ่านก่อน go-live)

      

        
1. URL ของ flow = ความลับ
มันฝังใน JS บนเว็บ public ใครได้ URL ก็ยิงได้ · โทเค็นในฟอร์มเป็นด่านชะลอเท่านั้น → ยกระดับด้วย Cloudflare Worker คั่นกลางเพื่อซ่อน URL+token

        
2. PDPA — ข้อมูลอ่อนไหว
ชื่อ ทะเบียน ผลตรวจ = ข้อมูลส่วนบุคคล → จำกัดสิทธิ์เข้าถึง SharePoint list/log และโฟลเดอร์ Cloudinary · ตั้ง retention · อย่าเปิดอ่านผ่าน public key

        
3. Cloudinary unsigned preset
ใครมี cloud name + preset ก็อัปได้ → จำกัด format/ขนาด เปิด moderation แยกโฟลเดอร์

        
4. รหัสผ่านหน้านี้
เป็นการตรวจฝั่ง client (ดู source เห็น hash ได้) กันคนทั่วไป ไม่กันผู้เชี่ยวชาญ · การป้องกันจริงอยู่ที่สิทธิ์ของ SharePoint/Supabase
